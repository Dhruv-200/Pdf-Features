# PDF Studio

A privacy-first, client-side PDF toolbox built with Next.js and React. All PDF processing happens entirely in your browser - no uploads, no accounts, no tracking.

## âœ¨ Features

- **Merge PDFs** - Combine multiple PDF files into a single document with custom ordering
- **Split PDF** - Split a PDF into multiple parts using page ranges or fixed-size chunks
- **Page Manager** - Reorder, rotate, or delete pages with visual thumbnails
- **Images to PDF** - Convert JPG or PNG images into a PDF with configurable orientation and size
- **Extract Pages** - Create a new PDF containing only selected pages or ranges
- **Extract Text** - Extract selectable text from non-scanned PDFs with improved precision
- **Page Numbers** - Add page numbers to headers or footers with flexible formats
- **PDF to Images** - Convert PDF pages to PNG or JPG images
- **Rotate All Pages** - Rotate all pages by 90, 180, or 270 degrees
